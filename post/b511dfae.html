欧几里得算法

2020-09-16 算法 学习 评论

欧几里得算法，又叫做辗转相除法，是用来解决求两个数的最大公约数的经典算法。用gcd(a,b)来表示两个数的最大公约数。则此算法可以用递推式：

gcd(a,b) = gcd(b,a%b)

来表示。含义是，要求a，b的最大公约数，只需要求b和a%b的最大公约数即可。

证明
下面给出证明过程：

设 a = b * q + r，证明gcd(a, b) = gcd(b, r)。

由于gcd(a, b)为a，b的公约数，所以设a = m*gcd(a, b), b = n*gcd(a, b)。

r = a-b*q = m*gcd(a, b) - n*q*gcd(a, b) = (m-n*q)*gcd(a, b)

所以可得gcd(a,b)是r的因数。

由于gcd(b, r)为b，r的公约数，所以设b = x*gcd(b, r), r = y*gcd(b, r)。

a = y*gcd(b, r) + x*q*gcd(b, r) = (y+x*q)*gcd(b, r)

所以可得gcd(b, r)是a的因数。

因为gcd(a, b)同时是r和b的因数，所以gcd(a, b)是b和r的公因数（但不一定最大），所以有：

gcd(a, b) <= gcd(b, r)

同理可得：

gcd(b, r) <= gcd(a, b)

通过两式联立可知：

gcd(a, b) = gcd(b, r)

证明完毕。

代码实现

由于给出了递推关系，那么递归的思路是最容易想到的

递归基是b等于0，此时a即为最大公约数

非递归则可以通过循环实现
